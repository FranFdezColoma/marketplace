## 📋 Descripción

Este plugin proporcionará a los equipos de desarrollo agentes especializados y skills que cubran el ciclo de vida completo en Salesforce: desde el diseño de arquitectura hasta el desarrollo pro code (Apex, LWC) y la gestión del ciclo de release con SFDX.

Compatible con **GitHub Copilot CLI** y **Claude Code**.

---

## 🚀 Instalación

```bash
/plugin marketplace add FranFdezColoma/marketplace
/plugin install salesforce@marketplace
```

O localmente:

```bash
claude --plugin-dir /path/to/salesforce
copilot --plugin-dir /path/to/salesforce
```

## 📦 Stack Soportado

| Categoría | Tecnologías |
|-----------|-------------|
| **Pro Code** | Apex, Lightning Web Components (LWC), Aura Components, SOQL, SOSL |
| **Low Code** | Flow Builder, Process Builder, Validation Rules, Approval Processes |
| **Datos** | Salesforce Object Query Language (SOQL), Bulk API, REST API |
| **DevOps** | Salesforce CLI (`sf`), Salesforce DX (SFDX), GitHub Actions, Jenkins |
| **Testing** | Apex Test Framework, LWC Jest |
| **IDEs** | VS Code + Salesforce Extension Pack |
